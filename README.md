# internet-login-map
Website login database capable of drawing a graph of user's presence on the net.

Example data can be found in data.yml

### Usage:
Usage: ./Cli.py  -a | -g | -h | -l | -d ID | -s STRING [-f FILE]  
Examples:  
./Cli.py -a  
./Cli.py -g  
./Cli.py -d 42  
./Cli.py -s bear  
./Cli.py -s bear -f database.yml  
./Cli.py -l  
./Cli.py -h
